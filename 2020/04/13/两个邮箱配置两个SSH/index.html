两个邮箱配置两个SSH

一般来说，一台电脑上配置两个SSH密钥是很正常的嘛，两个邮箱对应两个SSH.
假设原来在C盘中/Users/xxx/.ssh目录下已经生成了一个密钥对：

生成第二个key
接下来我们生成第二个ssh key：

ssh-keygen -t rsa -c “yourmail@gmail.com“
这里不要一路回车，我们自己手动填写保存路径：

Generating public/private rsa key pair.
Enter file in which to save the key (/c/Users/xxx/.ssh/id_rsa): /c/Users/xxx/.ssh/id_rsa_github
再按回车三次
这时我们返回电脑的C盘/Users/xxx/下id_rsa_github来区别原id_rsa。

完成之后，我们可以看到~/.ssh目录下多了两个文件，变成：
